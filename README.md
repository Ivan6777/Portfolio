# Portfolio Landing Page Design

A modern, high-performance portfolio landing page built with **React**, **TypeScript**, and **Vite**. This project features a sleek, dark-themed design with glassmorphism effects, smooth animations, and fully accessible UI components.

-   **Accessible Components**: Built using **Radix UI** primitives for robustness and accessibility.
-   **Fast Performance**: Blazing fast load times and HMR (Hot Module Replacement) powered by **Vite**.

## ğŸ Getting Started

Follow these steps to set up the project locally.

### Prerequisites

Ensure you have **Node.js** (version 18+ recommended) installed on your machine.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd Portfolio-Landing-Page-Design
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

### Development

Start the development server with hot reload:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the app.

### Build via Production

Build the app for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
